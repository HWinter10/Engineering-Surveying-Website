## CK3 Website Project

This project is a responsive static multi-page website built from scratch using HTML, CSS, and JavaScript. It demonstrates front-end development fundamentals including structured navigation, responsive layout design, and basic interactive features.

The site adapts to both desktop and mobile screen sizes using responsive CSS. All layout, graphics, and visual elements were designed as part of the project, and the codebase is organized to clearly separate HTML structure, CSS styling, and JavaScript functionality.

### Features
<ul>
  <li>Multi-page navigation structure</li>
  <li>Responsive design for desktop and mobile screens</li>
  <li>Shared styling across pages using CSS</li>
  <li>Image slideshow and botton reaction implemented with JavaScript</li>
  <li>Organized front-end project structure separating HTML, CSS, and JavaScript</li>
  <li>Custom visual layout and graphics designed specifically for the site</li>
</ul>

### Technologies
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
